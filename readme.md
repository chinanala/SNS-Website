# nowcoder社区项目

本项目是一个基于SpringBoot的社区平台。实现了邮箱注册、验证码登录、发帖、评论、私信、点赞、关注、统计网站访问次数等功能，数据库使用Mybatis、Redis，使用Redis构建异步消息队列，使用字典树过滤敏感词。具体可以参见项目总结和项目笔记。
